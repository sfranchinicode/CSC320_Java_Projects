# Module_4: Loop Constructs

## Program Description
This program reads a set of five floating-point numbers from the user. It calculates
and displays the total, average, maximum, minimum, and 20% interest on the total.

## How It Works
- The program prompts the user to enter 5 numbers.
- A `while` loop ensures exactly 5 inputs are read.
- A counter variable prevents an infinite loop.
- The program tracks: 
    - Running total
    - Maximum and minimum values
- After the loop ends, it calculates: 
    - Average
    - Interest

## Output
The program prints: 
- Total
- Average
- Maximum Value
- Minimum Value
- Interest on the total
