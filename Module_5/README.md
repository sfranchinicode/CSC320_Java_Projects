# Module 5: Monthly Temperature Tracker

## Program Description
This Java program stores and manages avaerage monthly temperatures for an entire year. It uses two arrays to store the months and their corresponding temperature values.

The program allows users to input a specific month to view its temperature. If the user enters "year", the program displays all monthly temperatures, the yearly average and identifies the highest and lowest temperatures. 

Key features include:
- Storage of months and temperatures using arrays
- User input to retrieve specific monthly data
- Yearly summary including average, highest, and lowest temperatures

## How It Works
The program initializes two arrays: one for the months of the year and one for their corresponding temperatures. The program prompts the user to enter a month or the word "year". 
If a valid month is entered the program searches the array for the month and displays the corresponding temperature. 
If the user enters "year":
- The program loops through all the months and temperatures
- Displays each month with its temperature, calculates 
- Calculates the yearly average
- Determines and displays the highest and lowest temperaturs

## Output
When the program runs it produces: 
- Temperature for a specific month 
OR 
- A full yearly report with statistics
